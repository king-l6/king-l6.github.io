1.4、新增打标Tab自动定位

需求

根据用户Gender属性，点击用户维度的打标按钮，自动定位到对应的tab

  1. 男性 => 天马-男性
  2. 女性 => 天马-女性

前端实现

第一步：编写Map对照数据

第二步：将接口api/ai-debug/user返回数据中的性别利用Map对象存到pinia中(此处删除了useHistoryUser.ts文件中的debounce，待考究)

第三步：拿出pinia中存储的gender值，过滤掉目前数组中的gender值，并把gender值添加到数组的首位

第四步：弹框右上角设置按钮弹框中重新排序确定后过滤掉目前数组中的gender值，并把gender值添加到数组的首位

1.5、新增数据同步功能

需求

  1. 点击同步按钮，会自动将卡片维度汇总到刷次维度。然后再正常提交即可
  2. 点击同步按钮，会自动将刷次维度汇总到用户维度。然后再正常提交即可

前端实现

加按钮绑定事件，当返回的data不为空的时候将data赋值给params.value.extra

(注意：此处由于后端返回的接口如果没有达到同步功能调用的前提就会返回状态码：500，所以要在axios的白名单中过滤一下接口)
