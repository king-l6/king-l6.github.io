node

Node.js发布于2009年5月，由Ryan Dahl开发，是一个基于Chrome V8引擎的JavaScript运行环境，使用了一个事件驱动、非阻塞式I/O模型，让JavaScript 运行在服务端的开发平台，它让JavaScript成为与PHP、Python、Perl、Ruby等服务端语言平起平坐的脚本语言。
Node.js对一些特殊用例进行优化，提供替代的API，使得V8在非浏览器环境下运行得更好，V8引擎执行Javascript的速度非常快，性能非常好，基于Chrome JavaScript运行时建立的平台， 用于方便地搭建响应速度快、易于扩展的网络应用。
NodeJS是一个JS运行环境，不是新的语言

必备知识-终端

每个系统都有终端，以windows系统为例，windows7之后默认有两个终端cmd和powershell，在我们日常使用中，cmd不容易报错。对于前端来讲没有任何区别
我们安装Git时自带了git bash这也是个终端。
所以咱们的电脑有三个终端cmd``powsershell``bash，用哪个都行。

打开终端

cmd

windows+R打开运行，然后输入cmd

回车即可

powershell

同样的方法，把cmd更改成powershell

怎么从powershell跳转到cmd

在powershell的界面 输入cmd回车即可

为什么会有这样的需求，前端的一些命令在powershell默认禁止执行，需要配置解开进制的操作。

终端路径

我们打开的终端，都是自动打开的路径的。未来我们需要用终端时，都是要在我们的项目中用的。我们在运行中打开的cmd，路径不是我们的项目。

我们想要跳转到我们的项目目录。如果能够直接打开就跳转到对应的目录最好。

如何在对应的目录中打开终端

终端中显示的路径需要是我们对应的目录的路径
在资源管理器中，打开项目目录，然后在地址栏里输入cmd回车


路径就是我们的项目路径。
注意！！！！打开终端一定要确认路径是否正确，再操作代码

掌握几个终端的基本命令

终端可以输入命令，在输入命令前，你得确保是终端环境。

dir

查看当前目录中所有的文件和文件夹

cd

进入到某个路径下

跳转到某个盘符

直接输入盘符:回车

执行记录中的命令

我们在终端执行过的命令都会被记录，按方向键 上 下 可以切换记录。

安装


下载后一直下一步安装即可。最后一步勾选，安装python等工具，一般都会失败。如果失败，自己手动安装python。

是否安装成功

打开终端，输入node -v回车，如果能正常看到版本，则表示安装成功。
继续尝试输入npm -v回车，可以看到版本，表示npm安装成功
