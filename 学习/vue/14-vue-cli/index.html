vue-cli

1.vue-cli脚手架

 vue.js有著名的全家桶系列，包含了vue-router，vuex， axios，再加上构建工具vue-cli（脚手架工具），就是一个完整的vue项目的核心构成。<br />vue-cli这个构建工具大大降低了webpack的使用难度，支持热更新，有webpack-dev-server的支持，相当于启动了一个请求服务器，给你搭建了一个测试环境，只关注开发就OK。

1.1 安装vue-cli

官网  –> 生态系统  –> Vue CLI

  1. 首先查看是否已安装过vue-cli
    vue -V
    //如果有的话，且不是需要的版本，就用下面命令卸载
    npm uninstall vue-cli -g
  2. npm i  @vue/cli -g
    //也可使用cnpm 来安装 ，比较快些 ( 安装cnpm的命令： npm i cnpm –g )
    cnpm install @vue/cli -g
    安装成功后，可用vue -V查看版本  安装完成之后输入 vue -V（注意这里是大写的“V”），如果出现相应的版本号，则说明安装成功。
  3. npm root -g 查看全局安装的位置

2.用vue-cli来构建项目

2.1 操作步骤

  1. vue create 项目名

或者
vue ui 通过可视化，按步骤完成项目创建

  1. 选择自定义配置 Manually select features
  2. 选择你需要的配置
    • Babel (必选)
    • TypeScript（项目中使用ts开发的话，就勾选，暂时先不选中）
    • Progressive Web App (PWA) Support (接口缓存，优化项目，不选)
    • Router （必选）
    • Vuex （必选）
    • CSS Pre-processors (css预处理器，必选)
    • Linter / Formatter (代码格式，不选)
    • Unit Testing (代码测试，不选)
    • E2E Testing（需求界面测试，不选）
  3. 4.根据你选的配置进行Y/N选择

2.2 对应图像效果

上下箭头键切换
空格键选择或取消


安装完毕：
cd 项目名
npm run serve (运行项目)
